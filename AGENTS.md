# Agent Rules

- When editing camera maker/model/profile lists, first verify the latest and most complete model coverage from current sources before writing code.
- Prefer official manufacturer pages, Adobe/Lightroom support information, or reliable current camera databases for model names and profile behavior.
- Do not rely only on memory for modern camera model lists, newly released bodies, or Lightroom profile compatibility.
- If a maker only has Adobe fallback behavior in Lightroom, state that clearly in code comments or user-facing notes instead of inventing a Camera Matching profile.
